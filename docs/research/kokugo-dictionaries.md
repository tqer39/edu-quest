# 国語辞典・公式資料リサーチメモ

## 文部科学省 教科書情報データベース（国語）

- 文部科学省が提供する検定教科書のデータベースで、学年別・教科別に教科書の本文や教材情報を検索できる。
- 小学校国語の教科書 PDF や付録資料を閲覧でき、語句の使い方を教科書通りに確認できる。
- URL: <https://textbook.mext.go.jp/elementary/japanese-language/>

## 文部科学省 小学校学習指導要領（国語）

- 全国の学校で使用される国語科の学習目標・内容・語彙指導の観点をまとめた公式ガイドライン。
- 低学年から高学年までの指導事項と学習活動例、語彙指導の考え方が記載されている。
- PDF: <https://www.mext.go.jp/content/20220810-mxt_kyoiku01-000005795_1.pdf>
- 概要ページ: <https://www.mext.go.jp/a_menu/shotou/new-cs/1384661.htm>

## 学研プラス 新レインボー小学国語辞典

- 学研プラスが提供する小学生向けの国語辞典。語義・用例・漢字情報をカラーで掲載し、QR コードから音声や動画などのデジタル特典にアクセスできる。
- 1～6年生までの学習漢字と語彙を網羅し、豆知識やイラスト解説で理解を促す。
- 特設ページ: <https://gakken-plus.co.jp/catalog/s-series/s02/s0201/>
